# README

* Requirements
  - Ruby 2.5.1
  - Rails 5.2.3
  - Yarn

* Run Following Commands:
  > bundle install

  > yarn install
  
  > rake db:create
  
  > rake db:migrate
  

* Start server 
  > rails s

* Tests available with RSPEC, RUN
  > rspec spec/\<DesiredTestTarget\>/\<TestName\>

  - availables TestTargets
    - models
    - requests
    - routing
